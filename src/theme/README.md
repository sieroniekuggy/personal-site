### Installation and set up:

1. Install [Node.js](https://nodejs.org)

2. Install the Gatsby CLI

   ```
   npm install -g gatsby-cli
   ```

3. CD to the forked repo

   ```
   cd personal-site
   ```

4. Install the packages

   ```
   npm install
   ```
   
5. Start the development server

   ```
   npm start
   ```
